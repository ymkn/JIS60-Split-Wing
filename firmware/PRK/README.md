## keymap.rb

この時点ではPRK FirmwareがPIOを用いた分割キーボードに対応していなかったため、左右個別にkeymap.rbを用意しています。つまり、本キーボードをPRK Firmwareで動作させるには左右それぞれにUSBケーブルを接続し、2台のキーボードとして認識させる必要があります。TRRSケーブルの接続には対応していないのでご注意ください。
